## Q52. 师兄/导师安排不想做的事情时，你怎么应对？

> "首先，我会区分'不想做'和'不应该做'。如果是合理但我不太感兴趣的任务（比如数据清洗、实验复现），我会调整心态，因为这些都是科研的必要环节，而且往往能从中学到新的技能。
>
> 如果我认为任务安排有不合理之处（比如与我的研究方向严重偏离），我会选择合适的时机与导师沟通，说明我的顾虑和建议的替代方案。关键是**带着解决方案去沟通**，而不是单纯拒绝。
>
> 在我的经历中，导师安排的数据治理工作虽然枯燥，但让我深入理解了医疗数据的质量问题，这反而成为了 MedIA 论文中数据清洗部分的核心经验。"

> 长期：推动 AI 在医疗领域的安全落地，让技术真正帮助医生和患者"

## 公式速查（面试前30分钟必看）

| 公式 | 用途 | 关键记忆点 |
|------|------|-----------|
| Attention(Q,K,V) = softmax(QK^T/sqrt(d_k))V | Self-Attention | 除以sqrt(d_k)，softmax前mask |
| LoRA: W' = W_0 + BA*(alpha/r) | 高效微调 | B初始化为0，scaling=alpha/r |
| KL(P\|\|Q) = ΣP*log(P/Q) | DPO/MP-Loss | log内是P除以Q，不是Q除以P |
| L_DPO = -log σ(β*log(π_w/π_ref,w) - β*log(π_l/π_ref,l)) | DPO核心 | σ是sigmoid |
| A_GRPO_i = (R_i - μ_R) / σ_R | GRPO优势 | 组内标准化 |
| RoPE: f(q,m) = q * e^(i*m*θ) | 位置编码 | 旋转矩阵，只依赖相对位置 |
